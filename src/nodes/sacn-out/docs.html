Sends DMX data for a single universe via sACN (Streaming ACN / E1.31).

Details

Each incoming message provides the DMX values for the configured universe, which the node transmits as an sACN sender. Depending on the speed setting the data is sent once per message or repeated continuously at the selected refresh rate.

DMX values in the payload are keyed by DMX channel, starting at key 1 (not 0). Depending on the Values setting they are expected as a percentage (0–100) or as raw DMX values (0–255).

Configuration

Universe number
The universe to send to, 1 to 63999. Default 1.
Source name string
Name of the sACN sender shown on the network (below 50 characters). Default Node-RED.
Speed number
Refresh rate in Hz: 0 (once), 24, 27, 30, 40 or 44. Default 0.
Priority number
Priority of the sender, 1 to 200. Default 100.
Values percent | absolute
Interpret the incoming payload as a percentage (0–100) or as raw DMX values (0–255). Default percent.
Blank output on stop boolean
Send an all-zero frame before closing so receivers go dark on stop/redeploy instead of holding the last look. Default false.
Interface string
IP address of the network interface used for sending. Leave empty to use the default interface.
Port number
Network port to send on. Defaults to 5568.

Inputs

payload array | object
DMX values as percentage, keyed by channel (1-based).
